# BruteForceCalling

## プロジェクトの概要

今すぐ連絡したい!!
でも電話してもなかなか繋がらない!!
そんなことにお困りの方に
BruteForceCallingに電話を番号を登録しておくと
相手が電話に出てくれるまで繰り返し電話をかけ続けてくれます。
そんなサービスです。
[BruteForceCalling](https://protopedia.net/prototype/801272ee79cfde7fa5960571fee36b9b)
